# Tienda-Autopartes-POS-CS

Sistema de gestión integral para una tienda de refacciones automotrices, desarrollado como proyecto final escolar. La aplicación permite administrar el inventario, realizar ventas, gestionar usuarios y ejecutar cortes de caja, todo bajo una interfaz gráfica amigable.

## Características Principales

* **Punto de Venta (POS):** Registro ágil de transacciones comerciales y generación de tickets.
* **Gestión de Inventario:** Control total sobre el stock, incluyendo carga de imágenes para productos.
* **Administración de Usuarios:** Módulo de seguridad para altas, bajas y cambios de personal con validación de credenciales.
* **Corte de Caja:** Herramienta para el balance de ingresos y arqueo de caja al final de la jornada.
* **Base de Datos Relacional:** Almacenamiento persistente utilizando Microsoft Access.

## Tecnologías Utilizadas

* **Lenguaje:** C#
* **Framework:** .NET Framework (Windows Forms)
* **Base de Datos:** Microsoft Access (`.accdb`)
* **IDE:** Visual Studio

## Estructura del Repositorio

* `Codigos Fuentes/`: Contiene la lógica del negocio y el diseño de los formularios.
* `Base de Datos/`: Incluye el archivo de base de datos `Proyecto_Final_BD.accdb`.
* `Imagenes Productos/`: Catálogo visual de las refacciones soportadas.
* `Iconos e Imagenes/`: Recursos gráficos utilizados en la interfaz de usuario.

## Instalación

1. **Clonar el repositorio:** `git clone https://github.com/ericbojado/Tienda-Autopartes-POS-CS.git`
2. **Abrir el proyecto:** Localiza el archivo `.sln` y ábrelo con Visual Studio.
3. **Ejecutar:** Compila y presiona F5 para iniciar desde el formulario de Login.
